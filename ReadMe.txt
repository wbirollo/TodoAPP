#Teste Desenvolvedor

##Projeto de cadastro de eventos

===========================

* Requisitos

Visual Studio

SQL Server

Postman ou Swagger



* Instalação

Extrair arquivos em um mesmo diretório

Restaurar Banco de dados a partir do arquivo Dados.bat

Abrir o projeto TodoAPP.sln

Alterar o Server em appsettings.json para a instancia do servidor local de SQL Server



* Como usar

Compilar usando Swagger ou Postman para testes

Para alterar o status de uma tarefa como concluída, digite o Id.

Para cadastrar uma nova tarefa, altere apenas os campos nome e data como no exemplo
